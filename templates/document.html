Graph structure by API

We also setup a programming interface access (api) to provide a reusable graph structure in json

Request Path : http://fmgrlab.com/api/hullwhite/

Request Parameter:

  • volatitlity (for example : volatitlity = 0.01
  • maturity (for example : maturity = 5)
  • alpha (for example alpha = 0.1
  • period (for example period = year
  • Interest rate : a list of interest rate :rate=0.0509389&rate=0.0579733


    • Sample

    http://fmgrlab.com/api/hullwhite/?volatility=0.01&alpha=0.1&maturity=4&period=year&rate=0.0509389&rate=0.0579733&rate=0.0630595&rate=0.0673464&rate=0.0694816

    To test the api, we had generated the interest rate calculated by: 0.08 - 0.05Exp(-0.18ti. According the publication of Hull and White (1994), this corresponds approximately to the U.S. term structure at the beginning of 1994

    http://fmgrlab.com/api/test/auto/
